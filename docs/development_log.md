# 开发与调试技术日志

## 技术难点：ROS 接口映射与 C++ 源码改造
官方 `unitree_guide` 默认通过键盘直接操作内存变量。为了让机器人具备自动化能力，我深入审计了 `IOROS.cpp`。
- **挑战**：最初尝试映射到 `userCmd` 结构体导致编译报错（类型不匹配）。
- **解决路径**：通过分析 SDK 头文件，确定运动指令应注入到 `userValue` 结构体的 `lx, ly, rx` 成员中。
- **成果**：成功打通了底层控制逻辑与外部 ROS 消息的壁垒。

## 仿真调试与参数优化
在执行正方形轨迹任务时，遇到物理引擎惯性导致转弯不足 90 度的现象。
- **调优方案**：通过多次实验，将转向时间常数从 2.6s 优化至 5.3s，并引入了 0.5s 的缓冲制动时间，显著提升了轨迹闭合精度。
